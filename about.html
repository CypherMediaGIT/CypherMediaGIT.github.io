How To Install Us

Project Cypher is now Available via Its own Repo

How to Install Project Cypher:
Open Kodi
•Select SYSTEM > File Manager
•Select Add Source
•Select None
•Type the following EXACTLY http://repo.run/cypherrepo and select Done
•Highlight the box underneath Enter a name for this media Source
& type cypherrepo if not already there
•Select OK
•Go back to your Home Screen
•Select SYSTEM
•Select Add-Ons
•Select Install from zip file
•Select cypherrepo
•Select plugin.video.ProjectCypher.zip
•Wait for Add-on enabled notification
Project Cypher 2.0.2 is now installed and ready to use.
As usual the add-on can be accessed via VIDEOS > Add-Ons > Project Cypher

Also as we now use IPTV that requires F4Mtester to be installed aswell... this can be installed from my repo same as you installed Project Cypher (this does not work for everyone), or you can install it from TVADDONS repo or the makers repo Shani (use google to find these)
Also we advise using a Maintenance tool regularly to clean kodi's Cache and every time you use the cache cleaning tool.. after you must force close Kodi by pulling the power cord out the back of your box or Raspberry PI, using CTRL ALT DEL if on PC or Application Manager on Android devices
